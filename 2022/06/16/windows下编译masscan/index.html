问：如何知道要填的为1929。而不是别的数字呢？

答：_MSC_VER是微软用来定义编译器主版本的宏定义。其中_MSC_VER == 1929 的1929 是要根据不同版本的VS替换的。

要去这个网址查：https://docs.microsoft.com/en-us/cpp/preprocessor/predefined-macros?view=msvc-170。

在最下方的表格处。或者直接页面搜索_MSC_VER

由于我是Visual Studio 2019 version 16.11。所以此处为1929。

最后点击”本地windows调试器”。

没有报错就是成了。
